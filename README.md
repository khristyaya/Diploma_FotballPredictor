# Diploma_FotballPredictor
Система аналізу, візуалізації та прогнозування результатів футбольних матчів, розроблена в рамках бакалаврської (кваліфікаційної) роботи.

## Про проєкт

Проєкт поєднує аналіз футбольної статистики, візуалізацію даних та прогнозування результатів матчів за допомогою методів машинного навчання.

Для дослідження використовуються дані трьох європейських чемпіонатів:

- Premier League
- Bundesliga
- Ligue 1

## Основний функціонал

- Аналіз статистики окремої команди
- Порівняння двох команд
- Прогнозування результатів матчів

## Використані технології

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Streamlit
- Plotly

## Моделі машинного навчання

- Logistic Regression
- Random Forest
- XGBoost
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

Найкращі результати продемонструвала модель XGBoost.
